Elements:-

  • It is one of the core component of html
  • It comsists of three combinations
    • Open tag
    • Content
    • Closing tag

    • Types:-

    1. Inline Elements:-

      • Elements are in a same line does not start in a new line
      • Height and width are not affected except img tag
      • Margin and padding are applicable for letf and right direction not for top and bottom

    2. Block Elements:-

      • Always starts in a new line
      • Height and width are affected
      • Margin and padding works in all directions

    3. Inline elements:-

      • To convert inline elements to block Elements
      • It works same as inline
      • Display="Inline block"
      • Ex:-heading tags,p-tags,pre tags
